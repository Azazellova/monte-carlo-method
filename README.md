# monte-carlo-method
Программа разработанная для вычисления статистической вероятности для 10000 имитаций

## Алгоритм программы

1. Генерация точек разлома:
      Случайная величина  X  моделировалась как равномерно распределённая на отрезке [0; 20]
   Для воспроизводимости результатов было установлено случайное зерно (np.random.seed(42)).
2. Вычисление площади прямоугольника
3. Проверка условия:
      Для каждого  S_i  проверялось, попадает ли площадь в интервал [36; 75]:
4. Подсчёт успешных испытаний
5. Вычисление статистической вероятности:

## Данные полученные программой
